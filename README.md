# Installation

### prerequisites
The ACM @ NCSU website is built using [Astro](https://astro.build/) and the [yarn](https://classic.yarnpkg.com/en/) package manager. Before cloning this repo you need to install [yarn](https://classic.yarnpkg.com/en/)

### Cloning and Development
* Clone the repository ```$ git clone https://github.com/nick-Sutton/acm_ncsu.git```
* Run ```$ yarn install``` in the project directory
* Run ```$ yarn build``` or ```$ yarn dev``` to view your changes
